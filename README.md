# octoarms-sdk

Unified SDK repository for Octoarms task runtimes.

## Layout

- `python/` - Python collector SDK (`collector_sdk`)
- `typescript/` - TypeScript collector SDK (`collector-sdk-typescript`)

## Python SDK

```bash
cd python
python3 -m unittest collector_sdk.data_api_test
```

## TypeScript SDK

```bash
cd typescript
npm test
```
